## Reporting Issues

Found a bug or have a feature request? [Open an issue](../../issues/new) and include:

- **Bug reports**: Steps to reproduce, expected vs actual behavior, browser/OS info, screenshots if applicable
- **Feature requests**: Description of the feature and why it would be useful
- **Data questions**: Package name, date range, and what looks incorrect

## FAQ

**Is this the same as pypistats.org?**
No. PyPI Stats builds on the work of [pypistats.org](https://pypistats.org) with additional analytics, dashboards, and features. We are not affiliated with pypistats.org.

**Where does the data come from?**
Download data comes from the public PyPI/BigQuery dataset, the same source used by pypistats.org.

**How often is data updated?**
Download data is updated daily. There may be a 1-2 day lag from BigQuery.

**Is there a free tier?**
Yes. Free accounts can explore package stats, view trends, and access the blog. Registered accounts get additional features like saved packages, alerts, and newsletters.
